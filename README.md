# Zen Jigsaw Master Online

A calm single-game Next.js site for embedding `Zen Jigsaw Master` and targeting intent-driven SEO pages.

## Quick start

```bash
npm install
npm run dev
```

## Before deployment

Set these environment variables in Vercel:

- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_CONTACT_EMAIL`

## Build

```bash
npm run build
```
